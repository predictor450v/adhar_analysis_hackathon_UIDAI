# Aadhaar Data Analysis - Exported Files

## Overview
This folder contains exported data from the Aadhaar Data Analysis project.

## File Descriptions

### Core Data Files
1. **enrollment_summary.csv** - Summary statistics for enrollment data
2. **enrollment_processed.csv** - Cleaned and processed enrollment data
3. **demographic_summary.csv** - Summary statistics for demographic updates
4. **demographic_processed.csv** - Cleaned demographic update data
5. **biometric_summary.csv** - Summary statistics for biometric updates
6. **biometric_processed.csv** - Cleaned biometric update data

### Aggregated Data Files
7. **top_5_states_enrollment.csv** - Enrollment data for top 5 states
8. **enrollment_monthly_aggregated.csv** - Monthly aggregated enrollment data
9. **state_wise_summary.csv** - State-level summary statistics
10. **state_distribution.csv** - State-wise distribution of records
11. **district_distribution.csv** - District-wise distribution
12. **age_distribution_by_state.csv** - Age group distribution by state

### Visualization Data Files
13. **enrollment_daily_trends.csv** - Daily enrollment trends
14. **enrollment_monthly_trends.csv** - Monthly enrollment trends
15. **correlation_matrix.csv** - Correlation between different metrics

### Report Files
16. **aadhaar_analysis_report.xlsx** - Comprehensive Excel report
17. **analysis_metadata.json** - Analysis configuration and software versions

## Usage Notes
- All CSV files use UTF-8 encoding
- Dates are in YYYY-MM-DD format
- Numeric values are formatted with commas for thousands
- Missing values are represented as empty cells

## Analysis Parameters
- Analysis completed on: 2026-01-19 19:19:43
- Data sources: Aadhaar API datasets
- Preprocessing: Duplicate removal, missing value handling, date standardization
